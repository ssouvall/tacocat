TacoCat: A Palindrome Checker


Check if a String is a Palindrome by Reversing it in JavaScript!


TacoCat is a JavaScript coding challenge that allows the user to input a string, then reverses that string and outputs the result. It then checks if the user's string is a palindrome.


What is a palindrome?: A palindrome is a word, sentence, or phrase that is the same forward and backwards. For example, the phrase "A man, a plan, a canal, Panama" is a palindrome because it spells the same phrase when read backward (with all spaces, special characters, and capitalizations removed). Another example is TacoCat, hence the title of this challenge!


Instructions


Completing this challenge requirs four basic steps.


First, all special characters and white spaces should be removed from the user's input. This is done in order to make the string comparison based strictly on alphanumeric characters. For example, "A man, a plan, a canal, Panama!" would falsly return as not being a palindrome if the spaces and exclamation point were not removed. This is accomplished by setting the user's input equal to string.replace(/[^a-z0-9]/gi,''). The regex code in the replace method checks the string and removes all characters that are not alphanumeric.


Second, the freshly scrubbed user input is set to lower case using the string.toLowerCase() method. Taking the input to lower case is important for the same reason as removing special characters and spaces: the computer would incorrectly miss a palindrome if one letter was capitalized. This should then be saved to a variable called finalInput.


Third, finalInput is reversed. This is accomplished using a reverse for loop. We loop through the string backward and saved the result to a new variable, so that the new variable is equal to the inputted string backward.


Finally, the finalInput variable is compared against the flipped version using an if/else statement. If they are equal, the input is a palindrome!
